# Tech Tour Presentation

High-tech interactive presentation built with Next.js, Three.js, and Framer Motion.

## Prerequisites

- Node.js 18+ installed

## How to Run

1. **Install Dependencies**
   ```bash
   npm install
   ```
   
2. **Start Development Server**
    ```bash
    npm run dev
    ```
   
3. Open Browser Visit http://localhost:3000

## Tech Stack
- Framework: Next.js 14 (App Router)
- Styling: Tailwind CSS
- Animation: Framer Motion (Magic Motion)
- 3D: React Three Fiber (R3F)
- Icons: Lucide React
